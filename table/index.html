表格 #

可配置型表格，支持更多自定义

按需引入 #

//main.js
import table from 'hzh-element-components/table'
import 'hzh-element-components/table/style.css'

app.use(table)

Api #

Props #

属性说明类型可选值默认值
options表格配置项TableOptions[]必填-
data表格数据any[]必填-
elementLoadingText加载文案string--
elementLoadingBackground加载背景颜色string--
elementLoadingSpinner加载图标名string--
elementLoadingSvg加载图标svgstring--
elementLoadingSvgViewBox加载图标svg的配置string--
editIcon编辑显示的图标string-Edit
isEditRow是否可用编辑行booleantrue / falsefalse
editRowIndex编辑行按钮的标识string--
pagination是否显示分页booleantrue / falsefalse
paginationAlign显示分页的对齐方式left / center / rightleft / center / rightleft
currentPage当前是第几页number-1
pageSize当前一页多少条数据number-10
pageSizes显示分页数据多少条的选项number[]-[10, 20, 30, 40]
total数据总数number-0

FormOptions数据格式 #

key说明类型
prop表格数据字段名string
label表格表头名称string
width表格单元格宽度string
align表格单元格对其方式left / center / right
slot自定义列模板的插槽名string
action是否是编辑选项boolean
editable单元格是否可编辑boolean

事件 #

事件名说明参数
confirm可编辑单元格确认事件scope: 当前行数据
cancel可编辑单元格取消事件scope: 当前行数据
size-change分页当前显示条数改变时触发的事件val: 当前页条数
current-change分页当前页改变时触发的事件val: 当前页

slots插槽 #

名称说明
cellEdit自定义可编辑单元格内容
editRow自定义可编辑行按钮内容
action自定义编辑区域内容
