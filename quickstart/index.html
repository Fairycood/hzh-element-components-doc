快速开始 #

因为我们当前组件库是基于element-plus二次封装的，所有项目当中必须确保已经安装和正确使用了element-plus，element-plus安装使用指南

使用指南 #

  1. 在项目目录安装hzh-element-components

    • npm安装：npm install -S hzh-element-components
    • cnpm安装：cnpm install -S hzh-element-components
    • yarn安装：yarn add -S hzh-element-components

  2. 全局引入

    在main.ts文件当中添加以下代码：

    import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'
import { hzhUI } from "hzh-element-components"
import 'hzh-element-components/style.css'

const app = createApp(App)

app.use(ElementPlus)
app.use(hzhUI)

  3. 按需引入

    import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'
import chooseIcon from 'hzh-element-components/chooseIcon'
import 'hzh-element-components/chooseIcon/style.css'
import chooseArea from 'hzh-element-components/chooseArea'
// ...

const app = createApp(App)

app.use(ElementPlus)
app.use(chooseIcon)
app.use(chooseArea)
// ...

  4. 使用

在需要的组件内参照以下文档了解具体使用方式
